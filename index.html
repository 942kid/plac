R package 'plac' version 0.1.0

A Pairwise Likelihood Augmented Estimator for the Cox Model under Left-Truncation

This is an R package to implement the semi-parametric estimation method for the Cox model introduced in the paper A Pairwise Likelihood Augmented Estimator for the Cox Model under Left-Truncation by Wu et al. (2016).

Installation

This package can be installed from github with the help of devtools:

# install.packages("devtools")
devtools::install_github("942kid/plac")
